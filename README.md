## 🎯 Descripción del Proyecto

Este proyecto consiste en el desarrollo de una **maqueta web (parte visual)** para una tienda de instrumentos musicales, enfocada en la exhibición de productos como:

- 🎸 Guitarras  
- 🎵 Bajos  
- 🎶 Otros instrumentos musicales  

La propuesta busca ofrecer una experiencia visual atractiva y una navegación clara para los usuarios.

---

## 🧩 Estructura del Sitio

La página está organizada en diferentes secciones:

### 🏠 Inicio
Presentación principal de la tienda con una introducción general.

### 🛍️ Instrumentos
Catálogo visual de productos disponibles.

### 🖼️ Galería
Sección dedicada a imágenes representativas de los instrumentos.

### 📞 Contacto
Formulario de contacto para la comunicación con los usuarios.

## ⚙️ Características Implementadas
